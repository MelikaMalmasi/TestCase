# TestCase
A repository containing test cases for a sample home page. Created as a personal QA workspace to document UI elements, flows, validations, and expected behaviors for homepage testing.

Overview

This test case verifies the main layout and key interactive elements of the website to ensure everything displays and functions correctly.

What Is Tested
- Top Container (Header): Visible, correctly aligned, responsive.

- Middle Content: Main content loads properly and displays as expected.

- Left Sidebar:

  + Login functionality

  + Brochure download via email

  + “Ask a Question” form

- Bottom Section: FAQ area appears and works (expand/collapse if available).

- Floating Button (Bottom Right): Visible and functional.
